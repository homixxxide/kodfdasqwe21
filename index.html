Проблемы с запуском?

Windows блокирует запуск установщика

Это может произойти из-за SmartScreen при первом запуске.

  1. Нажмите «Подробнее».
  2. Выберите «Выполнить в любом случае».
  3. При необходимости: Свойства файла → «Разблокировать» → Применить.
